Fundamental text and font styling

Fonts

Fonts have some properties, like color. Here's how to style fonts.

Font families

The font-family attribute sets a different font on your text. There are 5 generic names for fonts: serif, sans-serif, monospace, cursive and fantasy. Cursive and fantasy are inconsistent and should not be used.
There are web safe fonts. These are fonts that will display on any browser, theoretically. Here's a list:

Name Generic type
Arial sans-serif
Courier New monospace
Georgia serif

Font stacks

It's good practice to specify multiple font families, like the following. font-family: "Trebuchet MS", Verdana, sans-serif;

Font size

The font-size attribute specifies the size of your font. It is good practice to specify the <html> font size to 10px and everything else in rems, so that everything scales nicely.

Style, weight, transform, decoration

CSS provides 4 properties to alter the weight of text:

  • font-style: normal, italic or oblique;
  • font-weight: normal, bold, lighter, bolder. You can also specify a number from 100 to 900;
  • text-transform: none, uppercase, lowercase, capitalize, full-width;
  • text-decoration: none, underline, overline, line-through. You can also use multiple values, like underline red wavy.

Text drop shadows

The text-shadow attribute applies a shadow effect to your text, like this. It receives 4 values:

  1. Horizontal offset;
  2. Vertical offset;
  3. Blur radius;
  4. Base color.

Text layout

There are attributes you can use to affect text layout.

  • text-align: left, right, center or justify. The justify spec makes the text spread out so that all lines of text are the same width.
  • line-height: use numbers as values, like 1.6. Makes the text look nicer.
  • letter-spacing: use dimensions as values, like 4px.
  • word-spacing: same as letter-spacing, but for words.

There are lots of other properties for text so research them if you want to really style your text.

Styling lists

There are some CSS properties that are specific to lists.

Bullet styles

The list-style-type specifies the type of bullets for the list, ordered or not. Setting it to upper-roman, for example, leads to this:

  1. 1 cup plain unsweetened almond milk;
  2. 1/2 cup plain full fat Greek yogurt;
  3. 1 scoop vanilla protein powder;
  4. 1 frozen banana;
  5. 1/8 tsp ground cinnamon;
  6. ice as needed.

Bullet position

The list-style-position attribute specifies whether the bullets are displayed inside the items, or outside of them. The default value is outside. Here's what inside looks like:

  1. 1 cup plain unsweetened almond milk;
  2. 1/2 cup plain full fat Greek yogurt;
  3. 1 scoop vanilla protein powder;
  4. 1 frozen banana;
  5. 1/8 tsp ground cinnamon;
  6. ice as needed.

Bullet image

The list-style-image attribute adds an image as your bullet. However, you don't have much control over the bullet's positioning. It is better to use the background-image attribute and erasing the original bullets.

List counting

You can control the counting of ordered lists.

  1. start: add start="x" to the <ol> tag to make the list start at x;
  2. reversed: add reversed to the <ol> tag to make the list count down;
  3. value: add value="x" to the <li> tags to specify the exact list item value.

Styling links

It is important to understand how to make use of pseudo-classes to style link states effectively.

Link states

Link states can be styled by using different pseudo-classes.

  • Unvisited (:link): A link's default state;
  • Visited (:visited): A link when it is in the browser's history;
  • Hover (:hover): A link when the mouse pointer hovers over it;
  • Focus (:focus): A link when it is moved to by a keyboard;
  • Active (:active): A link when it is being activated.

Web fonts

The options are pretty limited when it comes to web-safe fonts. Luckily, there is a way to embed font files into your website. 

    
      @font-face {
        font-family: "myFont";
        src: url(myFont.woff);
      }

      html {
        font-family: "myFont", "Bitstream Vera Serif", serif;
      }
